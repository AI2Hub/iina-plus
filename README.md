# Install you-get

##### [Install you-get](https://github.com/soimort/you-get/blob/develop/README.md#installation) use homebrew:
```
brew install you-get
```
##### install ykdl
```
pip3 install ykdl
```
##### Check you-get's path with:
```
which you-get
which ykdl
```
##### Mark sure the output is:
```
/usr/local/bin/you-get
/usr/local/bin/ykdl
```
